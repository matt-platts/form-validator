Basic validation and auto-correction for form fields:

  • Validation routines return true or false for whether text entry passes the test.
  • Auto-correct will auto-correct the input dependent on the rules.
  • Initialised using _v prototype (_v.autocorrect(input,type)), or as an extension of the String object (string._autocorrect(type)).
  • Easy to extend and and new rules.
  • This page serves to demo the usage.
